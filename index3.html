F3. CSS

Delfråga 1: Vad är en regel i CSS och vad består den av?

CSS är ett språk för att beskriva utseendet på ett dokument, det vill säga att det är ansvarigt för hur webbsidor ser ut:

färgen på bakgrunden och dekorativa element, storleken och stilen på typsnitt.

Termen står för Cascading Style Sheets.

CSS-koddesignregler:

Det måste finnas ett semikolon i slutet av raden

Använd två mellanslag för att dra in regler

Betydelsen av färger reduceras inte

Allt är skrivet med små bokstäver

Inga nollor hoppas över

Använd dubbla citattecken

Mellanslag efter kolon

Mellanslag efter kommatecken i färger

CSS-språket har sin egen syntax. Den har regler - värden som bestämmer utseendet på element. En CSS-regel består av en väljare, CSS-egenskaper och deras värden:

Väljare är markörer som hjälper webbläsaren att förstå vilken del av HTML-koden som ska tillämpas på angivna parametrar.

CSS-egenskaper är specifika designparametrar, såsom färgen på ett element eller text (färg) eller bakgrundsfärg (bakgrund).

Ett värde är helt enkelt ett värde och uttrycks som text eller ett tal, till exempel svart.

Delfråga 2: Vad är skillnaden mellan id och class? När bör man använda id respektive class?

id

CSS, id (identifierare) används för att tillämpa stil på unika element på en webbsida. Det unika med ett element innebär att ett element med denna design endast kommer att användas på sidan en gång. Sådana element kan vara: sidhuvud, sidfot, navigeringsmeny, etc.

För att använda id-väljaren måste skapa en identifierare (id), ge den ett unikt namn och skriva det i id-attributet för det element som stilen ska tillämpas på. I CSS-kod börjar id-väljaren med en #-symbol omedelbart följt av namnet på identifieraren.

Varje identifierare kan endast visas en gång på en sida, dvs. ett specifikt id ska bara användas på en sida med taggen som den är avsedd för. Om samma identifierare appliceras på mer än ett element, för det första kommer HTML-koden inte att passera valideringen, och för det andra kan detta göra att webbläsaren felaktigt bearbetar koden och man kanske inte ser det förväntade resultatet.

class

CSS-Class låter, precis som id-väljaren, utforma ett specifikt sidelement, men till skillnad från id:t låter classväljaren tillämpa stil på flera element på en webbsida, inte bara ett.

För att använda classväljaren måste man ange vilket element på sidan vill man använda det på; för att göra detta behöver man bara lägga till class-attributet till HTML-taggen som man vill formatera och ange önskat classnamn som värdet.

I CSS-kod måste alla classväljarsymboler börja med en punkt, det är så här webbläsare känner igen classväljaren i CSS-formatmallen.

Endast bokstäver, siffror, bindestreck och understreck är tillåtna i classnamnet

Class namnet efter en punkt ska alltid börja med en bokstav

Class namn är skiftlägeskänsliga, till exempel kommer .Menu och .menu att behandlas som två olika classer i CSS

Skillnader mellan id och klass i css är:

id har högre prioritet i css än class.

ID måste vara unikt för hela html-dokumentet.

Class kan kombineras med andra Classer, separerade med ett mellanslag, men id kan inte.

id skapar ett objekt med samma namn i fönsterutrymmet, som kan nås via js.
